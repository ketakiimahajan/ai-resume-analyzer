# AI Resume Analyzer
An AI-powered resume analysis tool that provides ATS scores, improvement tips, and career guidance through an integrated chatbot assistant.

## Features
- **Resume Analysis** - Upload your resume and get instant feedback
- **ATS Scoring** - Check how well your resume passes Applicant Tracking Systems
- **Smart Tips** - Get actionable advice on tone, content, structure, and skills
- **Career Chatbot** - Ask questions about resumes, interviews, and job search
- **Puter Integration** - Secure cloud storage and AI processing

## Tech Stack
- React Router
- TypeScript
- Puter.js (Cloud storage & AI)
- Tailwind CSS
